Linear Regression

Introduction (0.5 คะแนน)

Linear Regression เป็นการศึกษาความสัมพันธ์ระหว่างข้อมูล 2 ตัวแปรเพื่อนำมาใช้ทำนายค่าตัวแปรตาม (y) โดยอาศัยค่าของตัวแปรอิสระเพียงหนึ่งตัว (x) ซึ้งทั้ง 2 ตัวแปรจะมีความสัมพันธ์กันในลักษณะเชิงเส้น ในการศึกษาเมื่อเรานำเอาชุดข้อมูลของตัวแปรทั้งสองมาพล็อตจุดข้อมูลลงบนกราฟ จากนั้นก็สามารถลากเส้นตรงเพื่อแสดงความสัมพันธ์ระหว่างจุดได้

Explain Algorithm (0.5 คะแนน)

สามารถใช้เส้นตรงแห่งความสัมพันธ์ของตัวแปรทั้ง 2 ตัวแปรมาทำนายหาค่าของตัวแปร y ด้วยค่าตัวแปร x โดยการลากเส้นจากค่าตัวแปร x ในแกน X ขึ้นตรงไปบรรจบกับเส้นความสัมพันธ์แล้วลากเส้นจากจุดบรรจบต่อไปตัดกับแกน Y และอ่านออกมาเป็นค่า y ก็คือผลของการทำนาย

วิธีคิดและคำนวณ

กำหนดสมการ: สมการของ Linear Regression คือ y = mx + b โดยที่:

  • y คือ ราคาบ้าน
  • x คือ ขนาดพื้นที่
  • m คือ ความชันของเส้นตรง (slope)
  • b จุดตัดกับแกน y (intercept)

คำนวณความชัน (m): สำหรับชุดข้อมูลนี้, เราสามารถคำนวณความชัน m จาก m =

คำนวณจุดตัด (b): หลังจากที่เราได้ความชัน m แล้ว, เราสามารถคำนวณจุดตัด b ด้วยสมการ b = ȳ - mx̄ โดยที่ x̄ และ ȳ คือค่าเฉลี่ยของ x และ y ตามลำดับ

ใช้สมการเส้นตรงในการทำนาย: ด้วยค่า m และ b ที่คำนวณได้, เราสามารถใช้สมการ y = mx + b เพื่อทำนายราคาบ้านสำหรับขนาดพื้นที่ใด ๆ

ตัวอย่างการคำนวณ:

สมมติว่าเรามีค่า m = 3 และ b = 0 , สมการของเราจะเป็น y=3x สำหรับบ้านขนาด 350 ตารางเมตร (x=350), ราคาบ้าน (y) ที่ทำนายได้จะเป็น:
y = 3×350 = 1050 ดังนั้น บ้านขนาด 350 ตารางเมตรจะมีราคาประมาณ 1,050 พันดอลลาร์

          import numpy as np

          from sklearn.linear_model import LinearRegression

          import matplotlib.pyplot as plt

          X = np.array([100, 150, 200, 250, 300]).reshape(-1, 1)

          y = np.array([300, 450, 600, 750, 900])

          model = LinearRegression()

          model.fit(X, y)

          X_predict = np.array([350]).reshape(-1, 1)

          y_predict = model.predict(X_predict)

          y_predict, model.coef_, model.intercept_
